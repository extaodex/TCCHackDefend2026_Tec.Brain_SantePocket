# Santé Pocket - Application Mobile (Patient)

Cette application est le **carnet de santé numérique** côté patient. Elle permet de scanner des documents, gérer ses vaccins et contacts d'urgence, et partager ses données avec le médecin via P2P.

Pour les instructions complètes, l'architecture et les détails du projet, veuillez consulter le [README racine](../README.md).
